# Image-Processing-Algorithms
# Практическая работа №2 Алгоритмы обработки изображений

## 📌 Описание проекта

Данная работа посвящена освоению базовых методов компьютерного зрения, включая:

- Предобработку изображений и фильтрацию шумов
- Сегментацию объектов
- Вычисление геометрических признаков

## 🛠 Используемые технологии

- **Python 3.x**
- **OpenCV** — загрузка, обработка, фильтрация, морфологические операции
- **NumPy** — работа с массивами, вычисления
- **Matplotlib** — визуализация
- **Scikit-image** — расширенные признаки формы

## 📊 Исходные данные

Три изображения:

1. Человек, указывающий на себя (на белом фоне)
2. Банда животных
3. Волшебная поляна в лесу (с добавленным шумом «соль и перец»)

## 🧠 Что реализовано

- Преобразование цветных изображений в оттенки серого
- Анализ гистограмм яркости
- Линейное преобразование контраста и яркости
- Гамма-коррекция
- Эквализация гистограммы
- Фильтрация шумов:
  - Усредняющий фильтр
  - Гауссов фильтр
  - Медианный фильтр
- Выделение границ:
  - Операторы Собеля, Прюитта, Робертса
  - Оператор Лапласа
  - Детектор Канни
- Пороговая сегментация:
  - Глобальный порог
  - Метод Оцу
  - Локальный порог
- Морфологические операции (эрозия, дилатация, открытие, закрытие)
- Выделение связных компонентов
- Расчёт геометрических признаков объектов
